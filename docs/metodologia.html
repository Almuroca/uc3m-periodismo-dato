Creación de esta página web

El objetivo de esta página web consiste en reunir todas las actividades realizadas durante el curso 2020/2021.

¿Cómo he realizado esta página web?

Antes de lanzar esta página web es necesario cumplir los siguientes pasos:

  1. Pandoc: para poder realizar este ejercicio he tenido que pasar todos los archivos que tenía en formato markdown a html. Para ello, tuve que instalarme Pandoc desde la terminal mediante el comando “sudo apt install”.
  2. Bootstrap: en la página de Bootstrap me descargué los ejemplos. Una vez estaban instalados utilicé el comando “unzip” para extraer los que a mi me interesaban que se encontraban en la carpeta assets/dist/. En esta carpeta había otras dos: js y css. La primera de ellas hace referencia al lenguaje java script y la segunda a la hoja de estilo que vamos a utilizar.
  3. Github carpetas: cuando ya tenía todo lo que necesitaba (Pandoc, ejemplos de Bootstrap) cree una nueva carpeta en mi repositorio uc3m-periodismo-dato. Para ello, desde la terminal me situé en mi repositorio mediante el comado (cd=change directory) y con el comando (mkdir=make directory) hice la carpeta docs/. Dentro de ella mediante el comando (cp=copy) introduje la carpeta css y js. En esta misma carpeta también copié la carpeta de imágenes que tenía, (cp ../uc3m-periodismo-dato/imagenes docs/.). Esta orden consiste en copiar la carpeta que tenía en mi repositorio a la nueva carpeta creada “/docs”.
  4. Github, pages: en el apartado “Settings” de mi repositorio me situé en la opción Pages. Ahí hay una ocpión que te permite establecer dónde se va a contruir tu página web. Por eso, yo elegí como branch la opción main y la carpeta /docs. Una vez elegí las opciones le di a guardar y se generó un link que me permite acceder a mi página web.

Pasos a seguir:

  1. Transformar todos mis archivos .md en .html: con comando Pandoc.
  2. Copiar y pegar el contenido html en mi index.html en la parte begin content.
  3. Modificar en el html algunas líneas. (16,36,84)
  4. Modificar ruta de imagen.
  5. Guardar con nombre diferente.

¿Cómo acceder a mi página web?

  • Atajo de teclado: control+o
  • Comando pyhton3 -m http.server para ver el puerto y el http de mi página web.
  • Github: te sitúas en “Settings” luego seleccionas Pages y te aparece el link en la parte superior.
